## Getting Started

To get a local copy up and running follow these simple example steps.

- `git clone` https://github.com/stanleySimeon/web-pricing/.git
- `cd web-pricing`
- `npm install`

## In your browser, go to

- `http://localhost:3000`

## To run the app

Type in your terminal

- `npm start`
